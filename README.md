## 📌 Project Overview

**CriminalIntent Android App** is a mobile application built in Java using Android Studio that tracks and logs “crimes” as digital entries. It was developed while learning Android fundamentals such as activities, fragments, persistent storage, and UI/UX design.

This project serves as a practical example of **local data persistence, fragment lifecycle management, and clean architecture in mobile apps**.

---

## 🧠 System Architecture

The app uses a **layered modular structure**:

### 🔷 Layers

- **UI Layer**
  - Activities & Fragments for displaying lists and details  
- **Repository Layer**
  - Interfaces between UI and data storage  
- **Data Storage**
  - SQLite Database for persistent crime records

## ⚙️ Core Functionalities

- 📅 Create & log crime records  
- 📌 View list of all crimes  
- 📝 View/edit crime details  
- 📸 Add photos to crime entries  
- 📍 Capture and display location  
- 🔄 Persistent storage using SQLite
